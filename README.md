# Installation GUIDE sunriseshop

#### python < 3.7  
### 1. pip install pipenv  
### 2. pipenv shell

# In Windows Terminal:
### cd world 
### python manage.py migrate  
### python manage.py createsuperuser
### python manage.py runserver
